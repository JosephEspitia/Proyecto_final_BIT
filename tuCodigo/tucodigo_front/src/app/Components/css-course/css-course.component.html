Iniciar Test

1. El primer punto consta de crear una clase que manipule el color de fondo de un contenedor de etiqueta DIV, la clase debe tener como nombre "Bg-color-div", el color de fondo debe de ser Rojo

2. El segundo punto consta de crear una clase que agregue un borde solido al contorno de un contenedor, la clase debe de tener como nombre "Border".

3. El Tercer punto consta de ajustar el ancho de un contenedor DIV a un 50 % de la pantalla, la clase debe tener como nombre "Ancho".

4. El Cuarto punto consta de ajustar la altura de un contenedor DIV a un 70% de la pantalla, la clase debe de tener como nombre "Alto".

5. El Quinto punto consta de agregar un padding solo en la parte de abajo a un contenedor DIV, la clase debe de tener el nombre de "Padding".
